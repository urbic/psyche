1.1. Installation

1.1.1. Obtaining Psylla

Pre-built binary packages for openSUSE, Debian and Ubuntu are available at openSUSE Build Service.

1.1.2. Building from Sources

The sources are available in the project repository hosted on GitHub.

To build a project, you first need to clone the repository. To do this, run the command 

% git clone https://github.com/urbic/psylla.git

After cloning the project repository, a new subdirectory named psylla is created in the current directory. Change to it: 

% cd psylla

1.2. Command Line Interface

1.2.1. Synopsis

psylla [option…] [--] [program [argument…]]

1.2.2. Command Line Parameters

options
--help, --usage, -h, -?
Show options summary.
--version, -V
Show the version of the interpreter.
--eval, -e expression
Evaluate an expression consisting of Ψ commands.
--classpath, --cp path
Additional extensions class path including directories and jar archives.
--librarypath, --lp, -I path

program
The name of a file containing Ψ program. The program is read from the standard input when the hyphen is specified. If the program is not specified, run REPL (read—evaluate—print cycle).
argument
Argument passed to Ψ program.
