The Psylla is an interpreted scripting programming language whose syntax is similar to PostScript®.

The Psylla interpreter is programmed in Java. It is possible to extend the language with both new operators and new types, as well as embedding the interpreter in Java applications.

This reference describes Psylla v0.10.0.
